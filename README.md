# Lab 5 - Firebase Firestore

Built on top of Lab 3 (ListyCity) code.

## Features
- Tap a city to edit it
- Long press a city to delete it
- Cities are synced with Firebase Firestore

## How to use
- Click the + button to add a city
- Tap any city to edit it
- Long press any city to delete it
- Click the menu (3 dots) top right for more info" > README.md
